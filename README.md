# naivebayes-datamining

how to install

    "git clone https://github.com/huzairuje/klorofil-withfrontend klorofil/"
    cd klorofil/
    composer install
    cp .env.example .env
    php artisan key:generate on command line
    php artisan serve
    happy coding.
